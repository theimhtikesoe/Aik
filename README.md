âœ¨ Core Features

Interactive 3D Background: The entire site floats on top of a live Spline scene. You can click, drag, and interact with the 3D model while navigating the content.

"Frosted Glass" UI: A clean, premium "glassmorphism" effect for all content panels, built with CSS backdrop-filter.

Smooth Scroll Animations: All content panels and the footer gracefully fade-in and slide-up on scroll. This is built with the native IntersectionObserver API for high performance.

Active Nav Highlighting: The navigation links ("About," "Projects") automatically detect which section is on-screen and highlight themselves.

Fully Responsive: Stacks clean. The desktop's 3-column layout seamlessly transitions to a single, focused column on tablet and mobile.

Developer Easter Egg: A professional "signature." Clicking the "AIK" logo in the nav triggers a frosted glass modal crediting the developer (you).
